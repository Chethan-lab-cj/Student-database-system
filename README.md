This Python program implements a simple student database system using a dictionary, where each student's name serves as the key and their USN along with marks in three subjects are stored as a tuple. The program contains records for 10 students and allows the user to search for a specific student by entering either their name or USN. Once a match is found through a case-insensitive comparison, the program calculates the average of the three subject marks and assigns a grade based on the average — 'A' for 90 and above, 'B' for 80 and above, 'C' for 70 and above, 'D' for 60 and above, and 'Satisfied' for anything below 60. Finally, it displays the student's USN, name, individual marks, average, and grade in a clean, readable format.
